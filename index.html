🔧 Firebase Setup Guide (One-Time)

To enable the cloud database so all team members share the same receipt data:

  1. Go to console.firebase.google.com → Create a free project
  2. Click Add App → Web, give it a name, copy the firebaseConfig object
  3. In the sidebar go to Firestore Database → Create Database → Start in test mode
  4. Open this HTML file in a text editor, find the firebaseConfig object near the top, and replace each REPLACE_WITH_… value with the real ones you copied
  5. Save and open — the banner will show "✅ Connected to Cloud Database"
  6. Upload the file to Netlify/Vercel/any host and share the link with your team

Until Firebase is configured, receipts are saved locally in the browser of whoever generates them.
